# 30分钟做一个抖音logo

今天我们来聊聊PS的使用，通过布尔运算做出抖音logo造型并结合图层类型关系做成炫酷的故障效果。

# 关于布尔运算

在logo设计过程中，布尔运算是避免不了的。即简单的造型通过布尔运算得出复杂的造型。这几天有朋友问我，为啥logo设计要用布尔运算呢？个人的理解是“规范性”和“可复用性”问题。通过布尔运算做出的图形，非常严谨、规整，可复用性很强~

# logo造型制作

好了废话不多说，下面开始教程。

A、利用椭圆工具画出484x484的正圆，填充颜色为#fe1955；然后ctrl+j复制一份图层把正圆缩小到216x216，两个圆的关系是水平垂直居中。然后ctrl+e合并两个图层，然后选择“直接选择工具”，再圆边上点击显示两个圆的路径（这一步非常关键，否则布尔运算不能使用）

B、接下来，点击属性栏的布尔运算选项。把“合并形状”改成“减去顶层形状”。

C、接下来利用辅助线确定圆的中心点位置

D、接下来画出一个134x519的矩形，位置如下。

E、再复制之前的正圆，宽高改成一个630x630，位置如下。

D、同理，再次通过布尔运算把多余的部分去掉即可。

E、接下来，画出一个70x294的矩形，再次利用布尔运算把正圆切掉一个口。

ok，logo造型就做出来了，是不是非常的严谨和规整呢！

# 故障效果制作

把所有图层合并成一个，再ctrl+j复制，颜色改成#24f4ee。位置往上、往左移几个像素（自己把握）。

接着，见证奇迹的时刻到了。把新图层的类型改成“线性减淡（添加）”，故障效果就出来了。

# css3实现

使用css3新增的一个混合属性，设置mix-blend-mode属性值为lighten即可。

mix-blend-mode: lighten;
